# Bulk-HL7-Message-Editor-
A Python-based tool for uploading, parsing, combining, and safely transforming HL7 v2.x messages at scale. Designed to support large healthcare data conversions with scoped edits and data integrity safeguards.

Why This Project Exists
Healthcare data conversions often require large-scale HL7 transformations that must be applied selectively and safely. This tool was built to enable controlled, auditable edits to HL7 messages while preserving message structure and minimizing risk during reprocessing.

### Tech Stack
- Python
- Streamlit
- HL7 v2.x
